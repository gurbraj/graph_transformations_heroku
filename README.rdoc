== README


A Ruby on Rails application utilizing JavaScript, HTML and PostgreSQL to create an interactive math education tool for high school students. The front end was built using Bootstrap and the application also includes TDD using RSpec. The visual transformations are made by sending JSON-objects from the server to the client.

source code: https://github.com/gurbraj/graph_transformations_heroku.
deployed app: http://graph-transformations-heroku.herokuapp.com/functions.
